# 智能衣橱穿搭规划

一个优雅的私人衣橱管理与穿搭规划 React 应用，帮助用户系统化管理衣物、智能搭配、日历规划和行李打包。

## 原始需求

> 请制作智能衣橱穿搭规划页面，React 页面围绕衣物照片、颜色、材质、季节、天气、场合、洗护状态、日历安排、行李清单和穿搭收藏设计。用户在手机上把上衣、裤装、外套、鞋包和配饰拖到搭配画布，选择通勤、约会、旅行、运动或面试后，页面提示温差、雨天、颜色冲突、正在晾晒、尺码不合和近期重复穿着。整体感觉应像一个有秩序的私人衣橱，而不是商品商城，衣物卡片、试搭区域、日期计划和替换建议要分区明确。
> 扩展天气改搭逻辑：当样例天气出现降温、阵雨、大风或强晒时，页面重新评估外套、鞋子、帽子和面料厚度，用户能看到某件单品被移除的原因，也能手动保留并接受舒适度提醒。
> 扩展洗护占用提示：衣物处于待洗、晾干、熨烫、送洗或借出状态时，拖进画布会出现柔和提示，并列出颜色、版型或正式程度相近的替代品；用户确认强行使用后，日历上要留下提醒。

## 项目简介

本应用围绕用户的私人衣橱场景设计，提供以下核心功能：

- **衣橱管理**：按上衣、下装、外套、鞋包、配饰分类浏览衣物，支持颜色、材质、季节、洗护状态多维度筛选
- **智能搭配画布**：支持拖拽或点击添加衣物到搭配区域，实时预览穿搭组合
- **场合选择**：通勤、约会、旅行、运动、面试五大场景适配
- **天气改搭逻辑**：降温、阵雨、大风、强晒四种极端天气条件下重新评估外套、鞋子、帽子、面料厚度，显示移除原因，支持手动保留并接受舒适度提醒
- **洗护占用提示**：待洗、晾干、熨烫、送洗、借出状态下拖入画布显示柔和提示，推荐颜色/版型/正式程度相近的替代品，强行使用后日历留下提醒
- **穿搭智能提示**：温差提醒、雨天提示、大风提示、强晒提示、颜色冲突检测、洗护状态预警、场合尺码适配、近期重复穿着提醒
- **替换建议**：针对有问题的衣物自动推荐同类替换选项，按颜色、版型、正式程度智能排序
- **日历规划**：将穿搭方案安排到具体日期，提前规划每日穿着，洗护提醒日历标记
- **行李清单**：为旅行打包创建衣物清单，跟踪打包进度
- **穿搭收藏**：保存喜欢的搭配组合，随时复用

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 6
- **样式方案**：Tailwind CSS 3
- **状态管理**：Zustand 5（带本地持久化）
- **拖拽交互**：@dnd-kit
- **路由**：React Router 7
- **图标**：Lucide React
- **日期处理**：date-fns
- **字体**：Playfair Display（标题）+ DM Sans（正文）

## 启动方式

### 前置要求

- Node.js >= 20
- npm >= 9

### 启动步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务

```bash
npm run dev
```

访问地址：http://localhost:5173

#### 3. 生产构建

```bash
npm run build
```

构建产物位于 `dist/` 目录。

#### 4. 预览生产构建

```bash
npm run preview
```

## Docker 一键启动（推荐）

### 前置要求

- Docker >= 20
- Docker Compose >= 2

### 启动步骤

#### 1. 构建并启动服务

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:3000

#### 2. 停止和清理服务

```bash
docker compose down
```

## 功能详解

### 天气改搭逻辑

当天气面板开启降温、阵雨、大风或强晒条件时，页面自动重新评估搭配画布上的衣物：

| 天气条件 | 评估范围 | 提示内容 |
|----------|----------|----------|
| 降温 | 外套面料厚度、鞋子保暖性、帽子保暖性、整体面料厚度 | 建议更换厚实外套、保暖鞋款、毛线帽，提醒面料偏薄 |
| 阵雨 | 鞋包材质、帽子材质、外套防水性 | 建议更换防水鞋款/帽款/外套，提示携带雨伞 |
| 大风 | 外套防风性、帽子稳定性、鞋型稳定性 | 建议更换修身防风外套、贴合帽款、贴合鞋款 |
| 强晒 | 帽子防晒性、上装面料透气性、鞋款闷热程度、外套必要性 | 建议添加遮阳帽、更换轻薄面料、移除过热外套 |

每条建议附带移除原因（如"降温：面料太薄不够保暖"），用户可点击"保留"按钮强制保留单品，此时显示舒适度提醒。

### 洗护占用提示

衣物洗护状态扩展为 6 种：

| 状态 | 标签 | 拖入画布行为 |
|------|------|-------------|
| clean | — | 正常添加 |
| washing | 待洗 | 弹出柔和提示，推荐替代品 |
| drying | 晾干中 | 弹出柔和提示，推荐替代品 |
| ironing | 熨烫中 | 弹出柔和提示，推荐替代品 |
| dry_cleaning | 送洗中 | 弹出柔和提示，推荐替代品 |
| lent_out | 已借出 | 弹出柔和提示，推荐替代品 |

替代品按颜色、版型（sizeFit）、正式程度智能排序。用户确认强行使用后，日历上会在对应日期留下提醒标记。

### 智能提示规则

| 类型 | 触发条件 | 级别 |
|------|----------|------|
| 温差提醒 | 日温差 > 12°C 或气温偏高/偏低 | info / warning |
| 阵雨提示 | 当日下雨且鞋包/帽子/外套材质易受损，或未携带雨具 | info / warning |
| 大风提示 | 大风天气外套不防风、帽子易被吹走、鞋型宽松 | info / warning |
| 强晒提示 | 强晒天气帽子防晒不足、面料过厚、外套过热 | info / warning |
| 降温提示 | 降温天气外套/帽子/鞋子保暖不足、面料偏薄 | info / warning |
| 颜色冲突 | 红绿、橙粉等传统冲突配色组合，或荧光色搭配 | warning |
| 洗护状态 | 衣物处于待洗/晾干/熨烫/送洗/借出状态 | error / warning |
| 尺码适配 | 面试场合穿宽松款、运动场合穿修身款 | info / warning |
| 重复穿着 | 近 3 天内穿过同一件衣物 | warning |

### 颜色主题

采用温暖的大地色系配色，营造私人衣橱的舒适感：

- 主背景：象牙白 `#FAF8F5`
- 强调色：赤陶色 `#C4735B`
- 文字色：炭灰色 `#2C2C2C`
- 辅助色：沙色 `#E8E2DA`、暖灰 `#8B8580`

## 注意事项

- 应用使用浏览器 LocalStorage 持久化数据（Zustand persist），清除浏览器数据会导致衣橱和搭配记录丢失
- 衣物照片使用 AI 图片生成服务，首次加载可能需要一定时间
- 目前为前端 Demo，数据为模拟数据，未接入后端服务
